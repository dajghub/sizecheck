# Chrome Web Store — Fiche SizeCheck

## Nom
SizeCheck — Convertisseur de pointures

## Description courte (132 caractères max)
Trouve ta taille Nike, Adidas, Converse et 5 autres marques en un clic. Détection automatique sur les pages produit.

## Description longue

SizeCheck vous aide à trouver la bonne pointure quand vous changez de marque de chaussures. Chaque marque ayant sa propre grille de tailles, les comparaisons sont difficiles. SizeCheck utilise les longueurs de pied en centimètres comme référence commune pour vous donner l'équivalence exacte.

Sur une page produit d'une marque supportée, l'extension détecte automatiquement le site et affiche votre taille correspondante en un clic. Vous pouvez aussi utiliser la popup depuis n'importe quelle page pour comparer toutes les marques disponibles.

Aucune donnée personnelle collectée. Votre préférence de marque est sauvegardée uniquement en local sur votre appareil.

Disponible aussi sur sizecheck.fr

## Catégorie
Outils

## Langue principale
Français

## Justification de la permission "storage"
Utilisée uniquement pour mémoriser la marque et la pointure habituelle de l'utilisateur entre les sessions, afin d'éviter de re-saisir ces informations à chaque visite.
